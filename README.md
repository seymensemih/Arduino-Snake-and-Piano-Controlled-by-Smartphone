# Arduino Snake and Piano Controlled by Smartphone

Parts used in the project;
1-Buzzer
2-2x16 LCD Display
3-10k Potentiometer
4-Arduino UNO R3
5-HC05 Bluetooth Serial Module
6-22 pcs Jumper connection cable
