# Arduino Snake and Piano Controlled by Smartphone

Parts used in the project;<br/>
1-Buzzer<br/>
2-2x16 LCD Display<br/>
3-10k Potentiometer<br/>
4-Arduino UNO R3<br/>
5-HC05 Bluetooth Serial Module<br/>
6-22 pcs Jumper connection cable<br/>
